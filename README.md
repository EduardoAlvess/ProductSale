# ProductSale

Api para gerenciar produtos e pedidos.

## Pré requisitos
- Docker

## Instruções para executar
1. Execute o comando 'docker-compose up -d'
2. Execute o comando 'docker container ls' e pegue a porta do container com o nome de 'product-sale-api'
3. Acesse no navegador 'localhost:{porta do container}/swagger/index.html' para ver a lista de end-points
